Selamat Datang di Filkom Apps

Situs ini merupakan tugas akhir mata kuliah Pemrograman Platform Khusus yang diberikan Dosen kami bapak Nurizal Dwi Priandani, S.Kom. untuk membuat website dengan api dari filkom apps. Api dari filkom apps yang akan dipakai adalah api berita.

Semua isi artikel berdasarkan Api yang dijadikan acuan tetapi tata letak dan design web murni pembelajaran dari mata kuliah yang sudah ditempuh serta bantuan dari w3schools.com mengenai components yang tidak ada pada mata kuliah sebelumnya.

Pembuatan Web ini menggunakan text editor Atom yang memungkinkan lebih cepat memahami syntax-syntax karena warna syntax dari tag, class, atribut dll bervariasi, serta tidak menggunakan Adobe Muse CC karena menurut kami pribadi itu hanya dapat mengasah kreativitas kita dalam mendesign menggunakan selection atau tool-tool yang sudah tersedia, jika menggunakan text editor biasanya kita akan mengeja syntax yang ingin digunakan dan akan terbiasa jadi dapat dipahami serta diingat.

Terimakasih untuk Bapak Nurizal Dwi Priandani, S.Kom. sudah mengajar selama 1 Semester ini di kelas TIF T mata kuliah Pemrograman Platform Khusus. Semoga ilmu yang sudah diberikan dapat bermanfaat kelak dan digantikan dengan pahala.
